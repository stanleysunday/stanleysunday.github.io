# Lluvia de Fotos

Aplicación web interactiva: cada click o toque en la pantalla suelta una lluvia de fotos con física realista (gravedad, rebotes, fricción).

## Demo

Abre `index.html` en cualquier navegador.

## Hosting con GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages**
3. En Source selecciona **main** y carpeta **/ (root)**
4. Tu app estará en `https://tu-usuario.github.io/rain-photos/`
